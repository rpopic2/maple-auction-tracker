메이플 옥션 트래커 v.1.0 by 스카니아 seauma

빈칸에 명령어를 입력하면 됩니다.

> 빠른 도움말
에디셔널큐브 5400만
에디셔널큐브 5500만
에디셔널큐브 5200만
그래프 에디셔널큐브
이렇게 입력하면 변화하는 그래프를 볼 수 있습니다!


> 데이터 추가 방법
{아이템 이름} {가격} [날짜]: 아이템의 가격 등록. 아이템 이름에 뛰어쓰기 X!
ex) 에디셔널큐브 5400만
ex) 레인디어의창 9600
이렇게 입력하면 데이터세트에 가격 정보가 등록됩니다!
항상 지금 날짜와 시간으로 기록되므로 주의해주시기 바랍니다.
ex) 1.5억
가격만 치면 마지막으로 추가한 아이템의 가격으로 자동 등록됩니다.

지원하는 숫자 단위 키워드 : 만, 억, 조. 그냥 1.5치면 15000으로 입력됩니다.

> 데이터 보는 법
보기 : 전체 데이터세트를 봅니다.
보기 {아이템이름} : 아이템의 가격표를 봅니다.
그래프 {아이템이름} : 아이템의 가격 그래프를 봅니다.
그래프 : 마지막으로 본 아이템의 그래프를 봅니다.

> (고급) 데이터 세트 관리
초기화 {서버이름} {캐릭터이름} : 빈 새로운 데이터 세트 생성. 서버, 캐릭터 노출이 싫으면 아무거나 입력해도 됩니다.
불러오기 {.csv파일} : 같은 폴더 안의 다른 데이터세트를 불러옴

> 데이터 삭제나 수정을 원하면 해당 .csv파일 (폴더 안에 있습니다)을 찾아 삭제하시면 됩니다.
