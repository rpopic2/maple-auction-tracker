

빈칸에 명령어를 입력하면 됩니다.

> 빠른 도움말
에디셔널큐브 5400만
에디셔널큐브 5500만
에디셔널큐브 5200만
그래프 에디셔널큐브
이렇게 입력하면 변화하는 그래프를 볼 수 있습니다!


> 데이터 추가 방법
[아이템 이름] {가격} [날짜]: 아이템의 가격 등록. 아이템 이름에 뛰어쓰기 X!
ex) 에디셔널큐브 5400만
ex) 레인디어의창 9600
이렇게 입력하면 데이터세트에 가격 정보가 등록됩니다!
ex) 1.5억
가격만 치면 마지막으로 추가한 아이템의 가격으로 자동 등록됩니다.

지원하는 숫자 단위 키워드 : 만, 억, 조. 그냥 1.5치면 15000으로 입력됩니다.

날짜는 입력하지 않으면 현재 시간으로 기록되며 시간, 날짜, 년도를 모두 입력할 수 있습니다.
ex) 블랙큐브 4300만 11:30 -> 오늘 11:30분
ex) 카르마의가위 1억3000만 11/20 -> 올해 11월 20일 00시 00분


> 데이터 보는 법
보기 : 전체 데이터세트를 봅니다.
보기 {아이템이름} : 아이템의 가격표를 봅니다.
그래프 {아이템이름} : 아이템의 가격 그래프와 시간 순으로 정렬된 표를 봅니다.
그래프 : 마지막으로 본 아이템의 그래프와 시간 순으로 정렬된 표를 봅니다.
보기 명령은 시간 순 정렬이 아니고 그래프 명령은 시간 순 정렬입니다.

> (고급) 데이터 세트 관리
새파일 {서버이름} {이름} : 빈 새로운 데이터 세트 생성. ex) 새파일 스카니아 0
불러오기 {서버이름} {이름} : 같은 폴더 안의 다른 데이터세트를 불러옴 ex)불러오기 스카니아 0

> 데이터 수동 관리
데이터 삭제나 수정을 원하면 해당 .csv파일 (폴더 안에 있습니다)을 찾아 삭제하시면 됩니다.
맨 윗줄에 있는 정보는 지우시면 안됩니다. 전부 다 지우고 싶으면 파일 전체를 지우시기 바랍니다.
