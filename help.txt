

빈칸에 명령어를 입력하면 됩니다.

> 빠른 도움말
에디셔널큐브 5400만
에디셔널큐브 5500만
에디셔널큐브 5200만
그래프 에디셔널큐브
이렇게 입력하면 변화하는 그래프를 볼 수 있습니다!


> 데이터 추가 방법
[아이템 이름] {가격} [날짜]: 아이템의 가격 등록. 아이템 이름에 뛰어쓰기 X!
ex) 에디셔널큐브 5400만
ex) 레인디어의창 9600
이렇게 입력하면 데이터세트에 가격 정보가 등록됩니다!
ex) 1.5억
가격만 치면 마지막으로 추가한 아이템의 가격으로 자동 등록됩니다.

지원하는 숫자 단위 키워드 : 만, 억, 조.
* 1억 5천이라 입력하지 말고 1.5억이라 입력해야합니다! 나중에 추가할 예정.

날짜는 입력하지 않으면 현재 시간으로 기록되며 시간, 날짜, 년도를 모두 입력할 수 있습니다.
ex) 블랙큐브 4300만 11:30 -> 오늘 11:30분
ex) 카르마의가위 1억3000만 11/20 -> 올해 11월 20일 00시 00분


> 데이터 보는 법
전체 : 입력 순서으로 정렬된 전체 데이터세트를 봅니다.
{아이템이름} : 시간 순으로 정렬됨 아이템의 가격과 그래프를 봅니다.
* 아이템 이름을 입력 하고, 비교할 다른 아이템 이름을 입력하면 두 아이템이 모두 그래프에 나타납니다. 그래프 창을 닫으면 초기화됩니다.
그래프지우기 : 그래프를 지웁니다.
지우기 : 가장 마지막 줄에 있는 데이터를 지웁니다.

> 홈 화면
홈 화면에는 현재 등록되어 있는 물건들의 마지막 시세를 볼 수 있습니다.
빈 칸에 엔터를 치면 홈 화면으로 돌아옵니다.

> (고급) 데이터 파일 관리
새파일 {서버이름} {이름} : 빈 새로운 데이터 세트 생성. ex) 새파일 스카니아 0
불러오기 {서버이름} {이름} : 같은 폴더 안의 다른 데이터세트를 불러옴 ex)불러오기 스카니아 0

> 데이터 수동 관리
데이터 삭제나 수정을 원하면 해당 .csv파일 (폴더 안에 있습니다)을 찾아 삭제하시면 됩니다.
맨 윗줄에 있는 정보는 지우시면 안됩니다. 전부 다 지우고 싶으면 파일 전체를 지우시기 바랍니다.

> 단축키
Enter : 명령 입력
Ctrl-d : 종료
Ctrl-l : 마지막에 본 아이템 그래프 제외하고 모두 지우기
<위 화살표> : 이전 명령 불러오기 (아직 1개까지만 가능)
